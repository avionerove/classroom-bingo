# 🎯 3x3 빙고 교실 앱

선생님이 빙고 스티커를 만들고, 학생들이 각자 빙고판에 스티커를 배치한 뒤, 선생님이 정한 규칙(한 줄/두 줄/세 줄/풀하우스)대로 빙고를 완성해 **빙고를 외친 순서대로 선생님 화면에 이름이 뜨는** 교실용 실시간 웹앱입니다.

추가 설치 없이 **맥에 기본 내장된 Python 3** 만으로 동작합니다. (서버·DB·인터넷 불필요)

## 실행 방법

1. 터미널을 열고 이 폴더로 이동:
   ```bash
   cd ~/Documents/Claude/Projects/bingo
   ```
2. 서버 실행:
   ```bash
   python3 server.py
   ```
3. 터미널에 표시되는 주소를 사용합니다:
   - **선생님 화면**: `http://<선생님컴퓨터IP>:8000/teacher`
   - **학생 입장**: `http://<선생님컴퓨터IP>:8000/`

   > 포트 8000이 이미 사용 중이면 자동으로 8001, 8002 … 로 바뀝니다.
   > 실제 표시되는 주소를 그대로 쓰세요.
4. 학생들은 **선생님과 같은 WiFi**에 연결한 뒤, 위 '학생 입장' 주소를 휴대폰/노트북 브라우저에 입력하고 이름을 적어 입장합니다.

종료: 터미널에서 `Ctrl + C`

## 진행 순서

### 선생님
1. **① 빙고 방식 선택 & 스티커 만들기** — 먼저 **빙고 방식**을 고릅니다.
   - **일반** — 입력한 단어/숫자를 그대로 부르는 전통 빙고. (칸 하나만 입력)
   - **학습 (문제 → 답)** — **문제를 먼저** 쓰고 **칸에 들어갈 답**을 입력합니다. 예) 문제 `7×8` / 답 `56` → 학생 화면엔 "7×8"만 뜨고, 학생은 "56" 칸을 찾습니다. (수학·영어 단어·사회 용어 등)
   - 답은 9개 이상 권장. 방식은 게임 중간에 바꿔도 호출 내용이 그에 맞게 바뀝니다.
2. **② 규칙 선택** — 한 줄 / 두 줄 / 세 줄 / 풀하우스. **⏳ 제한시간**도 선택.
3. **📝 배치 단계 시작** — 학생들이 스티커를 빙고판에 끌어다 배치합니다. (모래시계 타이머)
4. **▶ 게임 시작** → **🎲 문제 부르기** 카드가 열립니다.
   - **다음 문제 무작위로 부르기** 버튼을 누르거나, 목록에서 직접 골라 부릅니다.
   - 부른 문제는 큰 글씨로 표시되고(선생님 화면엔 정답도 함께), 학생은 **불린 항목만 색칠**할 수 있습니다.
5. 학생이 빙고를 누르면 **🏆 순위**에 누른 순서대로 이름이 뜨고, **색종이+효과음** 축하 연출이 나옵니다.
   - 학생 현황표에서 **리치(한 칸 남음)** 도 실시간으로 보입니다.
6. **↻ 색칠 초기화(재시작)** — 배치 유지하고 호출만 초기화. **⟲ 전체 초기화** — 준비 단계로.

### 학생
1. 이름을 입력하고 입장.
2. 배치 단계가 되면 스티커를 빈 칸으로 **드래그**(또는 탭). '자동 채우기(섞기)'도 가능.
3. 게임이 시작되면 화면 위 **"이번에 불린 문제"** 를 보고, 빙고판에서 **해당하는 답 칸을 클릭**해 색칠합니다.
   - 아직 안 불린 칸은 색칠되지 않습니다.
4. 한 칸만 남으면 **🔥 리치!**, 규칙을 만족하면 **🎉 빙고!** 버튼으로 외칩니다.

## 특징 (종이 빙고 대비 디지털만의 장점)
- **앱이 문제를 무작위로 호출** + 중복 방지 + 호출 기록 → 공정하고 진행이 깔끔.
- **학습 빙고**: 문제(단서)와 정답을 분리해 수학·영어·사회 퀴즈 빙고로 변신.
- **자동 판정·순위**: 빙고 성립을 **서버에서 검증**해 누른 순서를 정확·공정하게 기록.
- **실시간 리치 표시 + 축하 연출(색종이·효과음)** 으로 긴장감과 몰입감.
- 새로고침해도 이름·배치·색칠이 유지됩니다(브라우저별 고유 ID 저장).
- 선생님 화면에서 학생별 접속 상태·배치·색칠·완성 줄 수를 실시간으로 볼 수 있습니다.

## 참고
- 포트를 직접 지정하려면: `PORT=9000 python3 server.py`
- 실시간 동기화는 long-polling 방식이라 별도 라이브러리가 필요 없습니다.
